# Learn2Gether App - by *Stiliyan Yanev №2401321018*
## Concept
Learn2Gether is based on open course learning from different sources and people. Every individual has the chance to learn new skills in his preference or create new courses to share his knowedge with new students.
## Features
Learn2Gether App has the followings:
+ User system for Register as a new one or Login/Logout in an existing one
+ Free choice of unique username and Student/Teacher option for either sharing knowedge or learning new skills
+ List of all available courses and their descriptions, posted by Teachers
+ Access to enrolling to courses as a Student
+ Access to saving courses to a wishlist for later enrollment
+ Access to watch video based lectures to enrolled courses as Student
## Admin Panel
Admins have access to:
+ Check every user signed in the app
+ Change roles of a user
## Instalation
Clone the repository of the project, open the project in IDE (Prefarably VS Code for FrontEnd / VS for BackEnd) and start both projects:
+ For FrontEnd project, write in Terminal: npm run dev
+ For BackEnd project, open the solution project and run it in VS
